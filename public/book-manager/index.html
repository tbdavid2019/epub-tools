讀墨書櫃整理

把散落各處的書，收進同一個書櫃

這個工具適合誰？

書散了一地的人。讀墨買幾本、Kobo 囤幾本，實體書櫃又放進一批。買到後來，哪些買過、哪些沒買過，自己也記不清。

偶爾看見一本想買的書，付完錢才想起來——是不是買過了？

如果你也是這種人，這裡能幫你把散落各處的藏書收在一起，讓重影現形、新生有位置入住。

怎麼用？

  1. 把 Bookmarklet 拖到書籤列
  2. 到讀墨書櫃頁面，按一下書籤
  3. 書單自動掃描帶回來
  4. 重複的書自動標出來，也可以搜尋、篩選、匯出 CSV

不需要輸入密碼，用你平常的方式登入讀墨就行。

書單只存在你的瀏覽器裡，不會上傳到任何地方。

連接書櫃

讀墨 Readmoo
  1. 把 [撈讀墨書櫃] 拉到書籤列，或點「加入書籤列」（建議用電腦操作）
    2. 撈讀墨書櫃
  2. 到 readmoo.com 登入，打開書櫃頁面
  3. 在書櫃頁面點擊書籤，自動捲動掃描所有書
  4. 確認數量後自動跳回，書櫃列表就會出現在下方，重複的書會標出來

不需要輸入密碼。什麼登入方式都行（Passkey、OTP、密碼）。

常見問題

Bookmarklet 是什麼？
一小段存在書籤裡的程式。在讀墨書櫃頁面按一下，就會自動捲動掃描所有書名，然後帶回這裡。不需要帳號密碼。
書單存在哪裡？
存在你目前這個瀏覽器裡（例如 Chrome），不會上傳到任何地方。換一個瀏覽器或清除瀏覽資料就會消失，建議匯出備份。
可以撈幾次？
想撈幾次都行。同一個平台重複撈，已有的書不會重複加入。
重複怎麼判斷的？
比對書名。空格、標點、副標題括號這些差異會自動忽略，盡量幫你把同一本書認出來。
