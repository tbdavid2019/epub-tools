怎麼訂閱電子書特價到我的日曆？

看到喜歡的書，當天自動跳通知提醒入手 — 不用再每天記得回來看。Windows、Mac、iPhone、Android 全部都能用，這頁告訴你要怎麼做。

先搞清楚兩種用法

用法 A：單本書「先加進日曆」

隨手加

看到一本想買但還沒空看的書，點書卡上的「加日曆」按鈕，那一天日曆會跳通知 — 適合「我先記著，下班再買」。

用法 B：整個訂閱「自動每天進日曆」

推薦

把整個特價清單訂閱起來，之後每天的特價自動進你的日曆，不用再手動加。可以選只追某個平台（Readmoo / 博客來 / Kobo），不會塞爆。

哪個裝置用哪個方法最順？

裝置 單本加日曆 訂閱（推薦）
Windows / Mac 桌面 Google 日曆 ✓ Google 日曆 ✓
iPhone / iPad（Safari） Apple 日曆 ✓ Apple 日曆 ✓（最順）
Mac（Safari） Apple 日曆 ✓ Apple 日曆 ✓
Android Google 日曆 ✓ 用瀏覽器版設定 ⚠️

Windows / Mac 桌面：用 Google 日曆

單本加日曆

  1. 到 電子書每日特價 頁面
  2. 找到喜歡的書，點書卡上的「加日曆 ▾」
  3. 選「Google 日曆」
  4. 跳到 Google 日曆預填頁，按右上角「儲存」

訂閱整個清單

  1. 到 電子書每日特價 → 訂閱專區
  2. 選想追蹤的平台（全部 / Readmoo / 博客來 / Kobo），按「一鍵加到 Google」
  3. 跳到 Google 日曆，會問「要新增此日曆嗎？」按「新增」
  4. 完成。之後每天的特價會自動進日曆。
更新頻率：Google 日曆通常每 12-24 小時拉一次更新，所以新特價可能晚一天才出現，這是 Google 的限制，不是 bug。

iPhone / iPad：用 Apple 日曆（最順）

單本加日曆

  1. 用 Safari 開 電子書每日特價
  2. 找到喜歡的書，點「加日曆 ▾」
  3. 選「Apple 日曆 / iPhone」
  4. 系統會跳出「將此活動加入行事曆」視窗，按「加入」

訂閱整個清單

  1. 用 Safari 開 訂閱專區
  2. 找到想追蹤的平台卡片，點「複製連結」
  3. 把連結貼到 Safari 網址列 → Enter
  4. 系統會跳「訂閱此日曆」視窗，按「訂閱」→「加入」
  5. 完成。打開「行事曆」App 就看得到。
注意：必須用 Safari 才會跳訂閱視窗。iPhone Chrome 會把連結當成檔案下載，不會自動進日曆。

Mac Safari 也是這樣

Mac 用 Safari 點訂閱連結 → 會自動開「行事曆」App → 跳「訂閱日曆」對話框 → 按「訂閱」即可。Mac Chrome 則會下載 .ics 檔，要自己打開檔案。

Android：用 Google 日曆

單本加日曆

  1. 用 Chrome 開 電子書每日特價
  2. 找到喜歡的書，點「加日曆 ▾」
  3. 選「Google 日曆」
  4. 會自動跳到 Google 日曆 App，按右上角「儲存」

訂閱整個清單

這裡有點麻煩 — Google 日曆 App 在手機上不能新增訂閱，是 Google 的長年限制。要繞一下：

  1. 先用桌面電腦或筆電（任何瀏覽器）打開 訂閱專區
  2. 按「一鍵加到 Google」 → Google 日曆網頁版會跳出新增對話框
  3. 用同一個 Google 帳號登入手機 Google 日曆 App，下拉重新整理
  4. 訂閱的日曆會自動同步出現在手機上
沒有桌面電腦？用手機 Chrome 開 calendar.google.com，**長按頁面 → 「桌面版網站」**，就能像電腦一樣新增訂閱。

常見問題

訂閱了之後，新特價多久會進日曆？

Google 日曆通常 12-24 小時拉一次新版，Apple 日曆預設 1 小時但可以調整。所以最久一天內會更新。如果想立刻看到新的，到日曆 App 裡找這個訂閱日曆，按「重新整理」或「立即同步」。

會塞爆我的日曆嗎？

每天大約 15-20 個事件（三平台特價加總）。如果只訂單一平台會少很多，大約 5-8 個。事件會在當天結束後自動消失（因為設成「全天事件」+ 透明顯示，不會擋你原本的行程）。如果還是覺得多，建議只訂自己常用的那個平台。

我訂閱了但看不到事件？

三個常見原因：
1. 日曆 App 還沒同步 — 等 1 小時或手動重新整理
2. 訂閱日曆預設可能被「隱藏」 — 到日曆設定把它打勾
3. iPhone 用 Chrome 訂閱會失敗 — 必須用 Safari

怎麼取消訂閱？

Google 日曆：左側「其他日曆」找到「電子書每日特價」 → 點旁邊的「×」
Apple 日曆（iPhone）：「行事曆」App → 底部「行事曆」→ 找到 HelloRuru → 點「資訊」→「刪除行事曆」
Apple 日曆（Mac）：左側列表上點右鍵 → 「取消訂閱」

會收到我的個資嗎？

不會。訂閱是你的日曆 App 主動來抓我們的公開連結（純文字 .ics 檔），就像你訂閱 RSS 一樣。我們不知道誰訂閱了，也不會記錄你的任何資料。

會花錢嗎？

完全免費。沒有付費版、沒有廣告、沒有任何隱藏費用。
