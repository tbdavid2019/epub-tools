  1. 選一個 TTF 或 OTF 字體檔（建議 SIL OFL 授權的免費字體）
  2. 選裝置預設，或自己填字級、寬高
  3. 填字體名（用在輸出檔名上）
  4. 按「開始轉換」，下載 .bin 檔
  5. 把 .bin 檔放進閱星曈裝置的字體目錄
