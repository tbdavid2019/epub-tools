TTF / OTF → .bin（閱星曈原裝） / .epdfont（風眼刷機版）

點這裡選檔案，或拖進來
支援 .ttf / .otf，純前端處理不上傳
32 px → 字格 32 × 42 px
關於鋸齒：閱星曈原廠 .bin 是 1-bit 黑白格式，鋸齒在物理上無法完全消除（風眼刷機版的 .epdfont 才能用 2-bit 灰階解決）。本工具預設「抗鋸齒對齊網格 + 閾值 100」會讓邊緣盡量平順；想再減輕鋸齒可選「字加粗 +1」讓筆畫變粗、相對鋸齒比例變小。
18 號 → 38 px @ 150 DPI（裝置實際渲染尺寸）
提示：本工具的 .epdfont 是依社群逆向版做的（header / metadata / bitmap 對齊），瀏覽器用 Canvas 渲染所以可能跟 Streamlit 版有像素級差異，裝置應該讀得到，建議先小批量測試。
提示：寬高由系統依字級自動套用，符合閱星曈裝置實際吃的尺寸。把轉好的檔案放進裝置的字體目錄即可使用。

畫面框模擬實際裝置尺寸（X3 = 528×792 / X4 = 480×800，橫版自動翻轉）。
字按裝置實際吃的 px 渲染，預覽會自動縮放配合畫面框。
  1. 選一個字體檔（.ttf / .otf，網路上找免費商用授權的，例如 SIL OFL）
  2. 選輸出格式（閱星曈原裝選 .bin、刷風眼選 .epdfont）
  3. 選字級（建議用官方常用字級快選）
  4. 填字體名（用在輸出檔名上）
  5. 按「開始轉換」，下載字體檔
  6. 把檔案放進裝置的字體目錄
授權提醒：請只轉換你有權使用的字體。商用請確認字體授權允許嵌入式使用。
