小提示
畫面框模擬閱星曈墨水屏（直版 750×1024 / 橫版 1024×750）。
字按裝置實際吃的 px 渲染，預覽會自動縮放配合畫面框。
  1. 選一個 TTF 或 OTF 字體檔（建議 SIL OFL 授權的免費字體）
  2. 選輸出格式（XTC 用 .bin / 風眼刷機版用 .epdfont）
  3. 選字級（建議用官方常用字級快選）
  4. 填字體名（用在輸出檔名上）
  5. 按「開始轉換」，下載字體檔
  6. 把檔案放進裝置的字體目錄
授權提醒：請只轉換你有權使用的字體。商用請確認字體授權允許嵌入式使用。
