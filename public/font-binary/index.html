  1. 選一個 TTF 或 OTF 字體檔（建議 SIL OFL 授權的免費字體）
  2. 選輸出格式：.bin（XTC 墨水屏）或 .epdfont（刷機版）
  3. 選字級（.epdfont 還可選字集大小）
  4. 填字體名（用在輸出檔名上）
  5. 按「開始轉換」，下載字體檔
  6. 把檔案放進閱星曈裝置的字體目錄
授權提醒：請只轉換你有權使用的字體。商用請確認字體授權允許嵌入式使用。
