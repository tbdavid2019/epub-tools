XTC Flasher

閱星曈 X3 / X4 繁中韌體刷機工具（MVP）

此工具需要 Web Serial API。 請用電腦版的 Chrome / Edge / Opera 開啟。 Safari、Firefox、手機瀏覽器都不支援。

使用前請注意

  • 插上 USB-C 線前，先按住 X3 的 BOOT 鍵，再按 Reset 鬆開，最後再放開 BOOT。
  • 刷機過程中請勿拔線、勿關閉分頁、勿讓電腦睡眠。
  • 建議先點「備份完整韌體」下載原廠 bin，日後想回復可以用。
  • 本工具是 HelloRuru 個人改作，非閱星曈官方工具，使用風險請自行評估。

① 備份完整韌體

讀取 X3 目前 Flash 整片 16 MB 內容，存成 backup.bin 檔。約 25 分鐘。

② 刷入繁中版韌體

刷入 HelloRuru 改作的繁中版（含最新 UI 修正）。約 2 分鐘。X3 / X4 不要選錯。

③ 還原備份（從自己的 bin）

選擇先前備份下來的 backup.bin（16 MB 完整韌體），刷回去。約 15 分鐘。

⚠️ 提醒：官方原廠 bin 自動下載的功能之後再補。現在先用「自己備份 → 自己還原」的方式，最安全。
